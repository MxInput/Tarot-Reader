### Features
- The ability to do a spread of up to ten cards.
- The option to include reversals.
- A save system for readings that includes the time it was saved.

### Downloading 
Get it here: https://github.com/MxInput/Tarot-Reader/releases

### How to Build
This project can be opened in Visual Studio and, from the Project tab, pressing Publish.
